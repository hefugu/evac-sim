# 火災工学・FDS連携の根拠メモ

## 目的とモデルの位置づけ

このシミュレータは、学校内の避難行動を扱うブラウザ上の避難シミュレーションである。
FDS/CFASTの考え方を参考にした低次元の火災・煙フォールバックを備えるが、Navier–Stokes方程式を解くCFDソルバではない。温度成層、圧力差、開口流、燃焼反応を詳細に解くものではなく、入力シナリオ間の避難挙動を相対比較するための近似モデルとして使用する。

詳細な火災安全評価では、外部のFDS等で解析した時刻別・セル別データをCSVとして読み込み、フォールバックより優先する。FDS CSVは本シミュレータ内の低次元モデルより高忠実度の入力として扱う。

## 低次元煙モデルの入力

フォールバックは、火災モデルが与える発熱速度 `Q [kW]` から燃料質量減少速度を次のように求める。

```text
燃料質量減少速度 [kg/s] = Q [kJ/s] / 燃焼熱 [kJ/kg]
すす発生速度 [kg/s]     = 燃料質量減少速度 × すす収率 × 発煙源倍率
CO発生速度 [kg/s]       = 燃料質量減少速度 × CO収率 × 発煙源倍率
```

すす、CO、上部煙層の体積、熱量は保存量としてセル間・階段間を移送する。天井ジェット相関を用いた水平移送、乱流拡散、隅角部混合、階段を介する上階移送、明示した換気口からの排出を組み合わせる。セル属性 `ventilation` は換気による一次除去率 `[1/s]` として扱い、開口流量 `[m3/s]` とは混同しない。画面上方を物理的な浮力方向として扱うことはない。

UIの初期値は次のとおりである。

| UI項目 / ID | 初期値 | 単位 | 意味 |
| --- | ---: | --- | --- |
| 発煙源倍率 / `smokeSourceRate` | 1 | - | すす・CO・煙層熱源へ掛けるシナリオ倍率 |
| 乱流拡散係数 / `smokeDiffusionRate` | 0.06 | m²/s | 同一階の未解像混合を表す有効拡散係数 |
| 天井ジェット倍率 / `smokeSpreadMix` | 1 | - | 天井ジェット相関で求めた移送速度の倍率 |
| 隅角部混合係数 / `smokeDiagonalMix` | 0.15 | - | 壁を貫通しない斜め方向混合の係数 |
| 一様漏気率 / `smokeDecayRate` | 0 | 1/s | 明示されていない一様な漏気を仮定する場合の一次速度定数 |
| すす収率 / `smokeSootYield` | 0.015 | kg/kg | 燃焼した燃料質量あたりのすす生成量 |
| CO収率 / `smokeCoYield` | 0.008 | kg/kg | 燃焼した燃料質量あたりのCO生成量 |
| 燃焼熱 / `smokeHeatOfCombustion` | 44,500 | kJ/kg | HRRを燃料質量減少速度へ換算する値 |
| 黄色出口の開放換気 / `smokeExitVent` | 無効 | - | 有効時だけ黄色出口を外気への常時開放口として扱う |

すす収率 `0.015 kg/kg`、CO収率 `0.008 kg/kg`、燃焼熱 `44,500 kJ/kg` は、名前付きの **n-ヘプタン・デモシナリオ**の値であり、すべての火災に適用できる材料定数ではない。実際の燃料、燃焼状態、換気条件に対応する実測値または根拠ある文献値を入力する必要がある。FDSやCFASTも、燃料のHRR・化学種収率等を利用者がシナリオ入力として定義する。

黄色出口は通常、避難先を表す編集記号であり、換気開口を意味しない。外気へ常時開放される扉・開口をシナリオとして仮定する場合に限り `smokeExitVent` を有効にする。未選択が既定である。

## 導出量

すす質量濃度から質量減光係数 `K_m = 8700 m²/kg` を用いて減光係数 `K [1/m]` を導出する。視認距離は `S = C / K` を基礎とし、反射標識を想定した `C = 3` を既定にする（自発光標識に用いられる `C = 8` ではない）。CO濃度は、煙層中のCO質量、煙層体積、空気密度からppmへ換算する。画面互換用の `smokeMap` はこれらの保存量から作る表示用指標であり、物理単位を持つ独立の保存量ではない。

区画上部へ達するプルーム量にはHeskestad型の巻込み相関、同一階の移送速度にはCFASTで説明されるAlpert/Heskestad型の天井ジェット相関を使う。放射分率は `0.35`、開口・スタック流の流量係数は `0.70` を既定とする。これらは文献に基づく既定値だが、区画形状や燃料に対する校正済み定数ではない。熱損失率 `0.012 1/s`、有効乱流拡散係数、隅角部混合係数は本低次元化のモデル仮定である。

各セルでは天井側の煙層体積から煙層下面高さを求める。避難者への曝露と経路危険度は床上 `1.6 m` の目線高さで評価し、煙層下面付近 `0.3 m` を滑らかに遷移させる。3D表示も固定高さではなく、天井と煙層下面の中点に描く。計算は描画頻度から独立した `0.1 s` 固定刻みを使う。ブラウザタブ停止などで長い空白が生じた場合は、全サブシステムを最大 `0.5 s` まで同じ刻みで進め、それを超える実時間はシミュレーションの一時停止として扱う。

煙層がセル容量を満たした場合、まず隣接する空き容量へ保存量を移送する。閉鎖されて移送先がない分については、新たなプルーム巻込み空気量だけを抑制・計上し、生成済みのすす、CO、熱量は消去しない。この満杯時処理も圧力を解くCFDの代替ではなく、閉鎖区画の絶対温度評価には使用しない。

## FDS CSV形式

以下の列を読む。

```csv
time_s,floor,cx,cy,heat_flux_kw_m2,extinction_coefficient_m_1,co_ppm,visibility_m,temperature_c
```

- `time_s`: 時刻 [s]
- `floor`: フロア番号。CSVでは1始まり、内部では0始まりへ変換する。
- `cx`, `cy`: evac-simのグリッドセル座標。
- `heat_flux_kw_m2`: 熱流束 [kW/m²]
- `extinction_coefficient_m_1`: 自然対数基準の減光係数 `K` [1/m]。
- `optical_density_base10_m_1`: base-10光学濃度の単位長さ当たり値。読み込み時に `K = OD × ln(10)` で変換する。
- 旧互換名 `optical_density_m_1` は従来どおり自然対数基準の `K [1/m]` と仮定して受理し、注意をログへ出す。意味が曖昧なため、新しいCSVでは使わない。
- `co_ppm`: CO濃度 [ppm]
- `visibility_m`: 視認距離 [m]
- `temperature_c`: 温度 [℃]

FDS CSVが読み込まれている場合は、該当する時刻・セル・物理量についてCSV値を優先する。空欄の物理量を0とは解釈せず、フォールバックで補う。時系列は現在時刻以前の最新値をセル別・物理量別にsample-and-holdし、未来のフレームを先読みしない。したがって、ある時刻のCSV行に変更されたセルまたは物理量だけを記載しても、それ以前に与えた別セル・別物理量の値は明示的に更新されるまで保持される。明示した `0` は有効な更新値である。これはFDS結果をブラウザ内で再計算する処理ではなく、外部解析結果を同じ `floor/cell/time` 座標へ重ねる処理である。入力側で単位、座標原点、階番号、時刻基準を検証しなければならない。

温度のみを入力したセルも、経路探索と曝露評価から除外しない。現在のスクリーニング用既定値は `60 ℃` を超えた分から経路ペナルティと温度曝露を加算し、`120 ℃` 以上を強い回避対象とする。これは完全な熱曝露・人体耐容モデルでも法令上の限界値でもなく、温度フィールドを無視しないための避難行動用しきい値である。熱流束、曝露時間、衣服、湿度等を含む設計評価では、対象基準に合わせて別途検証する必要がある。

## 導入根拠

- [NIST Fire Dynamics Simulator (FDS) manuals](https://pages.nist.gov/fds/manuals.html): FDSは火災による煙・熱輸送を扱う低速流向けLESコードで、Technical Reference、Verification、Validation等の文書が公開されている。
- [NIST CFAST Technical Reference, NIST TN 1889v1](https://doi.org/10.6028/NIST.TN.1889v1): 区画内を上部層・下部層として扱うゾーンモデルの質量・エネルギー収支、および火災・開口・煙輸送の技術的背景を示す。
- 避難安全ではASET/RSETの考え方により、視界、煙、有毒ガス、熱を避難可能時間の制約として扱う。
- t² fireは、火災成長を `Q = αt²` とする代表的なシナリオモデルである。

## 使用上の注意

- フォールバックの出力を、実建物の絶対的な煙到達時刻、温度、毒性または法令適合性の根拠にしてはならない。
- 煙・CO収率は燃料と燃焼状態に依存する。デモ初期値を対象建物の値とみなさない。
- 漏気、出口換気、階段の開放条件は建物シナリオに合わせて明示する。
- COのみの曝露量は、多成分ガス、低酸素、CO₂による呼吸増加を含む完全なFEDではない。
- 発表・報告では「FDS/CFAST-informed reduced-order fallback（低次元近似）」と「FDS CSVによる高忠実度入力」を区別して説明する。
- ResetおよびMonte Carloの各試行では、手動火元を初期強度・火災年齢0へ戻し、前試行で延焼したセルを消去する。同一シナリオの反復試行へ前試行の火災進展を持ち越さない。
