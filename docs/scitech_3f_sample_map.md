# 科学技術高校 3F サンプルマップ

このサンプルは、3F平面図を2D編集と2.5D/3D表示で共用する検証用マップです。

## 目的

- まず3Fを基準階として、火災・煙・避難行動を検証する。
- 2D画像から通行可能領域を抽出し、teacher / student / panic の行動モデル検証に使う。
- 2F・4Fを追加し、階段リンクで接続する際の基準にする。

## 元画像の扱い

元画像では以下の意味で色を使っています。

- 白: 廊下・通路
- 黒: 壁・部屋・通行不可領域
- 緑: 階段・外階段
- 文字: 室名・階段名などの注釈

サンプルデータは元の白・黒・緑を保持した正常なPNGをbase64化しています。読込時に4×4 pxを1セルとして白と緑を分類し、最大の4連結成分だけを残すことで、廊下から離れた白い文字を通路にせず除去します。緑は `walkable = true` かつ `stair = true` となり、右下の外階段だけは `outdoor`、その他は `indoor` として初期化されます。

上部廊下7セルを実寸2.9mへ合わせ、3F profileの初期値は `cellSizeMeters = 0.42` としています。校正の詳細は [`scitech_3f_3d_model.md`](./scitech_3f_3d_model.md) を参照してください。

## 使い方

1. `python -m http.server 8000` で起動する。
2. `http://localhost:8000/sim/` を開く。
3. 左パネルの `3Fサンプル読込` を押す。
4. 3Fが選ばれ、緑の階段候補が水色で表示されることを確認する。
5. 開始位置、出口、火元を配置する。
6. 上部の `3D` または `同時表示` を押して、同じ状態を3Dで確認する。
7. 必要に応じて危険度表示や煙パラメータを調整する。

`http://localhost:8000/sim/3d.html` は閲覧専用ページです。主画面が開いているとBroadcastChannel経由でライブ状態を受け取り、主画面がない場合は3Fサンプルを単独表示します。

## 推奨する初期配置

実験の出発点として、次のような配置が分かりやすいです。

- 開始位置: 下側廊下のHR3-1〜HR3-6付近
- 出口: 右下の外階段付近、左下階段付近、上側階段付近
- 火元: 中央廊下または右側縦廊下

## 次に検証すること

- 最短経路のみの避難と、火災リスクを考慮した避難の差
- teacher誘導あり/なしの差
- panic比率を変えたときの避難時間・死亡者数・スタック数
- 煙が階段やT字/曲がり角でどう広がるか

## 拡張

この3Fを基準として、次のデータを追加できます。

- 1F・2F・4Fなどの同寸法マップ
- 対応する階段セル間のリンク、種別、所要時間、容量
- FDS/CFD由来CSVの時系列データ
- フロアごとの換気・可燃性・ドア属性
