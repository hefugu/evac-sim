# 科学技術高校 3F サンプルマップ

このサンプルは、3F平面図を単一フロア避難シミュレーション用に前処理した検証用マップです。

## 目的

- まず1階層だけで、火災・煙・避難行動の検証を行う。
- 2D画像から通行可能領域を抽出し、teacher / student / panic の行動モデル検証に使う。
- 将来の2.5D/3D化では、この1フロアを複数階層へ拡張する基準にする。

## 元画像の扱い

元画像では以下の意味で色を使っています。

- 白: 廊下・通路
- 黒: 壁・部屋・通行不可領域
- 緑: 階段・外階段
- 文字: 室名・階段名などの注釈

現在のシミュレーターは画像の白色領域を通行可能セルとして抽出するため、サンプル読込用データでは、文字注釈を除去し、白い通路と緑の階段領域を通行可能領域として白色化しています。

## 使い方

1. `python -m http.server 8000` で起動する。
2. `http://localhost:8000/sim/` を開く。
3. 左パネルの `3Fサンプル読込` を押す。
4. 開始位置、出口、火元を配置する。
5. 必要に応じて危険度表示や煙パラメータを調整する。

## 推奨する初期配置

実験の出発点として、次のような配置が分かりやすいです。

- 開始位置: 下側廊下のHR3-1〜HR3-6付近
- 出口: 右下の外階段付近、左下階段付近、上側階段付近
- 火元: 中央廊下または右側縦廊下

## 次に検証すること

- 最短経路のみの避難と、火災リスクを考慮した避難の差
- teacher誘導あり/なしの差
- panic比率を変えたときの避難時間・死亡者数・スタック数
- 煙が階段やT字/曲がり角でどう広がるか

## 将来拡張

この1フロアを基準として、次の段階では以下を追加します。

- 階段セルの自動抽出
- 階段・吹き抜けを通じた上階への煙伝播
- 複数階層の2.5Dモデル
- 3D可視化ビュー
- FDS CSVとの階層データ連携
